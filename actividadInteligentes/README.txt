Para la ejecucion de este programa se necesita tener instalado python 3.
Se necesita ejecutar en un entorno como Visual Code y solo se debe correr el código
Para el ingreso de los datos manualmente:
Ingresar nodo por nodo y despues de cada nodo dar un Enter ("A")
Para pasar a asignar las conexiones de los nodos dar Ctrl+C
Ingresar los arcos de cada nodo en el siguiente formato: A,B
Dar enter y al acabar de ingresar dar Ctrl+Code

Para el ingreso de los datos por medio del archivo .txt:
Se escriben los nodos en la primera linea, separandolos por comas
Apartir de la segunda linea se ponen Vertice por nodo, cada uno de estos en una linea diferente
Al ejecutar el programa, el archivo .txt se debe encontrar n la misma carpeta del proyecto y se tiene que indicar en el nombre del archivo cuando lo solicite el programa
El orden indica el numero de nodos
El tamaño del grafo indica el numero de conexiones del grafo
