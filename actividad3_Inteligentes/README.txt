El programa necesita que se descargue la libreria Turtle,
https://pypi.org/project/PythonTurtle/
https://pythonturtle.org/
Descargar la última versión dependiendo su sistema operativo.

Ejecutar el programa, posteriormente ingresar las coordenadas del agente y de ahi empezara a moverse
o si se dio una coordenada fuera del rango, se le pedirá una coordenada correcta.

Ejemplos de coordenadas:

55,55
100, 120
-120, -60

NOTA: Dadas las propiedades del mundo en el que el agente puede habitar (el mundo de este agente se
limita a la figura), la tarea que tiene que desarrollar es seguir los bordes.
