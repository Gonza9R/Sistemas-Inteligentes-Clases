Para la ejecucion de este programa se necesita tener instalado python 3 asi como tener descargadas las librerias de pandas y openpyxl:
-pip install pandas
-pip install openpyxl
Ademas de tener el archivo de peliculas.xlsx en la carpeta del programa
Una vez que se cuenta con esto solo se necesita ejecutar el programa y ingresar los datos que se solicitan el cual es un numero para seleccionar la categoria

Descripción:
Implementa utilizando naive bayes.
Utilizando un archivo de texto de entrada para introducir al programa las categorías y los puestos de empleo. Por
ejemplo, puedes usar algo similar a esta estructura:
-Tecnología
Ingeniero de software
Full stack
Ingeniero de pruebas
...
-Ventas
Puesto 1
Puesto 2
...
Con esta información construirás tus modelos probabilísticos.
Posteriormente, de manera interactiva, proporcionarás el nuevo puesto que vas a clasificar. Permite
que el usuario pueda intentar clasificar la cantidad de puestos que desee antes de dar por terminado
el programa.
El programa debe mostrar las probabilidades que van generándose en el proceso de clasificación
hasta llegar al resultado
