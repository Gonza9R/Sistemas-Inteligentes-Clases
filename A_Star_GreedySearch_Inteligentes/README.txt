Para la ejecucion de este programa se necesita tener instalado python 3 asi como tener descargadas las librerias matplotlib.pyplot y networkx 
Ademas de tener los archivos cities_locations.txt (latitud y longitud de las ciudades) y ciudades.txt (es la distancia entre ciudades) 
Una vez que se cuenta con esto solo se necesita ejecutar el programa y ingresar los datos que se solicitan los cuales deben ser los nombres de las ciudades sin espacios y primera letra de cada palabra en mayuscula.
	-Ingresar la ciudad de origen 
	-Ingresar la ciudad destino
Posteriormente preguntara que metodo desea ocupar para lo cual puede elegir entre Gredy y A-Start. para buscar la ruta mas corta y de menor costo segun el tipo de busqueda.
